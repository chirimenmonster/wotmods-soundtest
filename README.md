# wotmods-soundtest

WoT の各種イベントに関連付けられた音声を再生する mod です。


## インストール

リリースページから wotmod ファイルをダウンロードして所定のフォルダにコピーしてください。
他の多くの mod と同様です。


## 使い方

F12 でイベントの選択ウィンドウが開くので、
soundMode, genderSwitch, nation, event
を選択してください。

play ボタンを押すと関連付けられた音声が再生されます。
